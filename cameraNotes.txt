The camera API provided by first does not provide access to all of the features of the camera.
This format is reverse engineered from a packet sniff of manually sending the settings from the web UI.
Multiple values can be set by a single request

Examples:
http://10.17.19.11/sm/sm.srv?root_ImageSource_I0_Sensor_ExposureValue=0&action=modify
http://10.17.19.11/sm/sm.srv?root_ImageSource_I0_Sensor_Contrast=0&action=modify
